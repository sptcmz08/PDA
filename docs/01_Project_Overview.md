# 01 Project Overview

## 1. ชื่อระบบ

PDA Tracking Scanner App

## 2. วัตถุประสงค์

พัฒนาแอป Android สำหรับเครื่อง PDA รุ่น iT88 Android 14 เพื่อใช้สแกนเลข Tracking จากใบปะหน้าขนส่ง เช่น J&T โดยมีเป้าหมายหลักคือป้องกันการสแกนเลขซ้ำ เพื่อช่วยลดความผิดพลาดในการแพ็กสินค้า เช่น ปริ้นใบปะหน้าซ้ำ ยิงเลขซ้ำ หรือแพ็กพัสดุผิดรายการ

## 3. ลักษณะการใช้งานจริง

พนักงานใช้เครื่อง PDA เปิดแอป แล้วกดยิงบาร์โค้ดบนใบปะหน้าขนส่ง

ระบบต้องทำงานทันทีหลังสแกน โดยไม่ต้องกดยืนยัน ไม่ต้องกด Enter และไม่ต้องเลือกเมนูหลายขั้นตอน

## 4. Workflow หลัก

```text
เปิดแอป
↓
แอป Focus ช่อง Scanner Input อัตโนมัติ
↓
พนักงานยิง Barcode / QR Code
↓
แอปรับค่า Tracking
↓
Validate ว่าเป็นตัวเลข 12 หลัก
↓
ตรวจสอบซ้ำใน SQLite
↓
กรณีเลขใหม่:
    บันทึกข้อมูล
    เล่นเสียง Success
    แสดงสีเขียว
    เพิ่ม Counter
↓
กรณีเลขซ้ำ:
    ไม่บันทึก
    เล่นเสียง Duplicate Alert
    แสดงสีแดง
    แสดงเวลาที่เคยสแกน
↓
กรณีรูปแบบผิด:
    ไม่บันทึก
    เล่นเสียง Error
    แสดงสีส้ม
↓
Clear ช่อง Scanner Input
↓
Focus กลับอัตโนมัติ
↓
พร้อมยิงรายการถัดไปทันที
```

## 5. Scope ที่ต้องทำ

ระบบต้องมีฟังก์ชันดังนี้

- สแกน Barcode / QR Code ผ่านหัวสแกนของ PDA
- รับค่าจากหัวสแกนแบบ Keyboard Wedge
- สแกนต่อเนื่องได้
- ไม่ต้องกดยืนยันทุกครั้ง
- ตรวจสอบเลข Tracking ซ้ำทันที
- เลขใหม่บันทึกอัตโนมัติ
- เลขซ้ำแจ้งเตือนเสียงและข้อความ
- เลขซ้ำไม่บันทึกซ้ำ
- แสดงจำนวนรายการที่บันทึกสำเร็จ
- ค้นหา Tracking ที่เคยสแกน
- ดูประวัติการสแกนพร้อมวันที่และเวลา
- Export CSV
- แชร์ไฟล์ผ่าน Android Share Sheet เช่น LINE / Gmail / Drive
- เริ่มรอบใหม่เพื่อล้างข้อมูล
- ใช้งาน Offline
- เก็บข้อมูลใน SQLite

## 7. ข้อมูลอุปกรณ์

- รุ่น PDA: iT88
- Android Version: 14
- Network: WiFi
- Scanner: Built-in Barcode / QR Scanner
- Input Mode: Keyboard Wedge / Keyboard Input
- พฤติกรรม Scanner: ยิงแล้วเลขเข้า Text Field อัตโนมัติ เหมือนใช้งานกับ BigSeller

## 8. รูปแบบ Tracking

Tracking ที่ถูกต้องต้องเป็น

- ตัวเลขเท่านั้น
- ความยาว 12 หลัก

```text
829360206500
829353044505
829357386314
829352691775
829361951095
```
